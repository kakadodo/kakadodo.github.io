註解

其實也沒甚麼好講的…平常用 CSS 寫註解的話應該會用 /* */ 這種符號，不過使用 Sass 的話，註解可以有兩種寫法:

  1. // ... 使用兩條斜線可以做單行註解
  2. /* ... */ 依然可以使用 CSS 的註解方式做單行或多行註解

不過 Sass 檔在編譯輸出後，可能會因註解使用的不同而有不同顯示。使用單行註解的話，編譯成 CSS 後註解會消失。使用多行註解編譯後會保留，但如果是進行編譯+壓縮的話則多行註解也會消失。只有在多行註解的開頭加上 ! 符號，才能保持註解始終存在。
